# Overview

![RIVER](./imgs/graphical_abstract.png)

Welcome to **RIVER**, a data platform for data analysis that seamlessly integrates **cloud storage (S3)**, **version control (Github)** and  **HPC (Slurm)** for scalable computing.
It will store the data from the **S3 bucket**, cloning the tools from the **Github repository**, running them on the **HPC** and put back the result to the platform. On the overview section, we will forcus more on the technical perspectives while the details on how to use it effectively and efficiently, follow **Usage** section
